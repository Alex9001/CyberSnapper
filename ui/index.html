🖥️ CLI Usage

node capture.js [urls.txt | url1 url2 ...]

Settings are loaded from config.json.

🔌 REST API

GET /api/screenshot?url=...&token=...
  • url: Target website (required)
  • format: png | webp | avif | pdf
  • token: API token from config.json (required)

🔧 Settings

  • Initial Delay — wait before scrolling (above-the-fold content).
  • Scroll Delay — wait between scroll steps (lazy-loaded content).
  • Final Delay — wait after scrolling back to top.
  • Concurrency — number of websites to capture in parallel.
  • Block Popups — block popups/modals via CSS + network rules.
  • Hide Selectors — CSS selectors to hide before capture (one per line).
  • Wait For Selector — wait for this selector before capture.
